# 🎓 VidyaPath — Your Personal AI Tutor

> *"Adaptive, patient, always there for you."*

VidyaPath is an AI-powered personalized learning platform built for students who learn at their own pace. 
Instead of one-size-fits-all content, VidyaPath first assesses what you already know, then teaches 
you accordingly — and keeps testing until you truly master the topic.

Built with a local LLM (Phi-3 via Ollama) and a clean Streamlit interface, VidyaPath works 
completely offline — no API costs, no data leaving your machine. It supports both English and Hindi, 
making it accessible to a wider range of learners across India.
## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Streamlit |
| LLM | Phi-3 (via Ollama) |
| Database | SQLite |
| Language | Python |
| Auth | SHA-256 Password Hashing |
